# fable.intermittent

Forecasting models for intermittent time series, extending the [fable](https://fable.tidyverts.org/) framework.

## Overview

`fable.intermittent` provides forecasting methods specifically designed for intermittent demand time series, where observations contain many zero values and non-zero demand occurs sporadically.

## Installation

You can install the development version from GitHub:

```r
# install.packages("remotes")
remotes::install_github("yourusername/fable.intermittent")
```

## Usage

```r
library(fable.intermittent)
library(tsibble)
```
